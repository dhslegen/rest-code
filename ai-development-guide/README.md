# AI辅助开发实战指南

> 从零到一，掌握AI辅助编程的核心方法论

## 项目简介

本项目旨在系统性地分享AI辅助开发的实战经验，涵盖LLM基础概念、主流AI编程工具（Cursor、Claude Code）的使用技巧、实战案例以及最佳实践。

## 文档结构

```
ai-development-guide/
├── README.md                 # 项目总览（本文件）
├── OUTLINE.md               # 详细提纲
├── CHANGELOG.md             # 变更日志
├── TODO.md                  # 待办事项
├── chapters/                # 各章节详细内容

│   ├── 02-ai-coding-tools.md
│   ├── 03-cursor-guide.md
```

## 快速开始

1. 查看 [OUTLINE.md](./OUTLINE.md) 了解完整的内容框架
2. 阅读 [TODO.md](./TODO.md) 查看当前进度和待完成事项
3. 参考 [CHANGELOG.md](./CHANGELOG.md) 追踪项目演进历史

## 目标受众

- 想要提升开发效率的程序员
- 对AI辅助编程感兴趣的开发者
- 正在使用或计划使用Cursor/Claude Code的用户

## 核心理念

> **不存在距离**，唯一的距离就是输入第一句话，提出第一个需求。
>
> **不存在难度**，理解本质，不拘泥于形式，从直抒胸臆、开门见山开始。
>
> **不存在银弹**，各种理论正在蓬勃发展，实践是检验真理的唯一标准。
